POSSE Dynamics

Dynamics is a comprehensive, repository-based framework for Progress 4GL developers who are building new applications, or who need to migrate existing applications to take advantage of new Progress technologies. The framework is designed for a distributed environment with full support for stateless Appservers and WebClient. Dynamics promotes and supports an n-tier architecture, with separate business logic, database access and support for various UI's including GUI and browser based client types.

POSSE Dynamics represents the current code base on POSSE. There are 2 active branches - the MAIN branch where all new development work is going on, and the POSSE_20 maintenance branch which is the stable branch built on Dynamics 1.1A with the latest bug fixes fixed through POSSE.

Progress Dynamics is the commercial version of POSSE Dynamics available from Progress Software Corporation. Please check out www.progress.com for more information on the commercial version.

Dynamics Site Number Allocator
To ensure that data integrity remains for each Dynamics installation, each site or location running Dynamics must be identified by a unique number worldwide. In addition, a unique site number is required for every instance/installment of the product. PSC hosts the Dynamics Site Number Allocation application so that you can reserve unique numbers for your Dynamics installation.
