Procedure Editor

The Procedure Editor is a color-coded source editor which is used to create, edit, compile and run unstructured 4GL procedure (as opposed to the structured procedures created by the AppBuilder). The Procedure Editor also provides functionality for checking syntax, managing multiple buffers, and customizing its environment.
