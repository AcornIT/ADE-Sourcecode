Method Libraries

The Method Libraries frame allows you to specify include files containing internal procedures and common code. This frame contains the following elements:

Submit
Submits your changes to the WebSpeed Transaction Agent.
 
Reset
Resets all fields to the values when the property sheet was displayed, or since the last Submit.
 
Edit Area
Allows you to enter the pathname for the include file. Place the name within curly brackets ({}).By convention, include files have the extension ".i". For example:

{src/web/method/html-map.i}

Note: You should enter only comments and include file references in the edit area. Normally, you should enter one include file per line.
