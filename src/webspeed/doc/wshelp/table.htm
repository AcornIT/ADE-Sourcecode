Table Template

Use the Table template to format your database results as an HTML table. This is not a self-standing HTML file. You must call it from a SpeedScript procedure that generates HTML.

Note that it has no HTML header information. Do not open this file in an HTML authoring tool that automatically adds HTML header and body tags, for example.

Customize the formatting and reporting options in this template. For example, to customize the reporting options, change the Preprocessor definitions (&SCOPED-DEFINE DB-TABLE and &SCOPED-DEFINE DB_FIELDn) to match the database tables and fields whose data you want to display.

For more information about the options provided in this template, refer to the Language Reference.
