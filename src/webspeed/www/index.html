WebSpeed Workshop

The main application development tool within the WebSpeed Workshop is the WebSpeed AppBuilder. The WebSpeed AppBuilder framework unifies traditional client/server and n-tier functionality with Web-based technology to support a broad, integrated range of application and development options. With the WebSpeed AppBuilder, you can quickly create complex application interfaces simply by mapping HTML and integrating database information. You can assemble applications using a variety of Web objects, ranging in scale from HTML interfaces to SmartDataObjects and CGI Wrappers.

The WebSpeed AppBuilder provides you with shortcuts for creating standard business objects through Progress SmartObjects. These are procedures that encapsulate standard user interface logic and data. The SmartDataObject automates development by letting you quickly and easily create reusable code for associating database information with user-interface objects.
