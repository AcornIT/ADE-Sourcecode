Progress Developer's Briefcase

The Progress Developer's Briefcase is a freely downloadable product from Progress Software Corporation that offers POSSE participants a way to build, compile, and execute the source code that is available on POSSENET. The Progress Developer's Briefcase is limited for development and testing purposes only.

NEW! 9.1Dx1 Briefcase

This is a new release of the Progress Developer's Briefcase, based on Progress Version 9.1D. There have been enhancements to the language to support V2 of Progress Dynamics. If you are currently working with the feature branch of POSSE Dynamics you should download this version in order to have a clean compile the code.
