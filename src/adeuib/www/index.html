AppBuilder

The AppBuilder is the primary visual programming tool in the Progress ADE. All of the work of fabricating and assembling application components is done with the AppBuilder. The AppBuilder's Section Editor adds convenience to editing 4GL code while preserving the standardized structure that works best with Progress application components, such as basic object and SmartObjects.
